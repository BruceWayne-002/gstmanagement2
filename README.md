# GST Management

A web-based GST Management portal for filing returns, managing invoices,
and tracking GST-related activities.

## Features
- Secure authentication
- GST return filing
- Dashboard overview
- Modular GST services

## Tech Stack

- TypeScript
- Vite
- Supabase
